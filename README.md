# HM-SDS - Secure Data Storage tool

Stores JSON in encrypted txt file.

Saves data on any edit command.

Type "help" to see how to use the commands.

# example
- pas
- `enter password`
- enc test `any data, for example, some funny lines`
- dec test
- => logs string above
